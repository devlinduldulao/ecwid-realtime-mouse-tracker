# Marketplace Assets

This folder contains the source artwork and screenshot captures used for the Ecwid marketplace listing.

Included assets:

- `app-icon.svg`
- `listing-banner.svg`
- `screenshots/dashboard-live.png`
- `screenshots/dashboard-preview.png`
- `screenshots/setup-snippet.png`

These screenshots should be refreshed from the current app UI before every submission or major visual update.

Commands:

- `npm run build`
- `python3 -m http.server 4273 -d dist`
- `npm run capture:marketplace`

Before submission, export or resize these files to the exact raster dimensions required by the current Ecwid publishing portal.
