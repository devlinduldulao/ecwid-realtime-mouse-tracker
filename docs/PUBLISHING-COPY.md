## App name

Realtime Mouse Tracker for Ecwid

## Short description

Merchant-only Ecwid dashboard for previewing cursor activity and running same-browser storefront friction checks without a database or always-on backend.

## Full description

Realtime Mouse Tracker for Ecwid gives store owners a lightweight way to inspect browsing friction from inside their own dashboard. The app includes a merchant-facing live overview, preview scenarios with realistic fake data, and a same-browser self-test flow that works without Redis, a database, or a persistent Node.js service.

This version is intentionally built for free static hosting. Merchants can host the app on GitHub Pages or another static provider, install the storefront helper snippet, and use the dashboard to inspect preview data or their own browser-local storefront testing session.

## Feature bullets

- Merchant dashboard inside the Ecwid admin environment
- Preview mode with realistic product, checkout, and campaign scenarios
- Same-browser live self-test for storefront journeys
- Browser-local bounded event storage with no server bill
- Install snippet generator for Ecwid custom JavaScript
- Static deployment workflow with CI and automated tests

## Limitation disclosure

This free static version does not include shared cross-visitor live aggregation. It supports preview mode and same-browser merchant self-tests only. Production-grade multi-visitor tracking would require a shared collector endpoint.
